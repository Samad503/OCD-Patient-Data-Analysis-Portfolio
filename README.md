# OCD Patient Data Analysis: End-to-End SQL & Power BI Project

This repository demonstrates a complete data analytics workflow—from raw data management using SQL to interactive storytelling and visualization in Power BI. The project analyzes a dataset of 1,500 OCD (Obsessive-Compulsive Disorder) patients to uncover clinical trends, demographic distributions, and symptom severity.

## 🚀 Project Workflow

- **Data Engineering (SQL)**: Used PostgreSQL/SQL to structure raw medical records, perform data cleaning, and execute complex queries to aggregate patient metrics.
- **Data Export**: Results from SQL queries were exported to CSV files for reporting.
- **Data Visualization (Power BI)**: Developed an interactive dashboard to visualize key performance indicators (KPIs) and clinical insights.

## 🛠️ Tech Stack

- **Database**: SQL (PostgreSQL) for data transformation and aggregation.
- **Visualization**: Power BI (DAX, Data Modeling).
- **Data Format**: CSV, SQL scripts.

## 📂 Repository Structure

- `Medical records.sql`: The core SQL script containing the database schema and the analytical queries used to generate the insights.
- `Ocd Analyst.pbix`: The interactive Power BI dashboard.
- `data/`: Contains the CSV exports generated by the SQL queries:
  - `Y-Bocs Score By Gender.csv`
  - `Ethinicity count and their average obssesion score.csv`
  - `people diagnosed with OCD MoM.csv` (Month-over-Month trends)
  - `most common Obsession Type (Count) ... .csv`
  - `most common Compulsion type (Count) ... .csv`

## 📊 Key Clinical Insights

- **Gender Neutrality**: The dataset shows a near-perfect split between Male (50.2%) and Female (49.8%) patients, with similar average obsession scores.
- **Severity Metrics**: The average Y-BOCS score (Yale-Brown Obsessive Compulsive Scale) for obsessions across the population is approximately 20, indicating a "Moderate" clinical severity.
- **Symptom Prevalence**: Harm-related obsessions carry the highest average severity (20.65). Washing is the most frequent compulsion type (321 cases), followed by Counting.
- **Temporal Trends**: Diagnosis rates were tracked from 2013 to 2019, showing steady clinical intake patterns over time.
